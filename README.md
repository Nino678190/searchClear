**searchClear** is a lightweight **browser extension** that helps you **clear or reset searches** easily, ensuring your searches remain clean and unaffected by previous input.

## üöÄ Installation

### Firefox

1. Download `searchClear-0.1.xpi`.
2. Open Firefox and drag the XPI file into the browser.
3. Confirm installation.
4. Set Masterpassword for Extension in Optionmenu

### Chromium / Chrome

1. Unzip `searchClear.zip`.
2. Open `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the unzipped folder.
5. Set Masterpassword for Extension in Optionmenu
